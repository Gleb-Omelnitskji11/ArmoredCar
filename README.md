# ArmoredVehicle

Гра в жанрі top-down, де гравець керує машиною та відстрілюється від зомбі. Проєкт побудований так, щоб було легко розширювати контент (додавати нових ворогів/турелі/кулі) без переписування логіки.

## Технології та підхід

- **Unity**
- **Zenject** (DI / інʼєкція залежностей, інсталяція сервісів та фабрик)
- **2D sprites**
  - Використовуються **atlases** та **slices** (нарізка спрайтів)
- **ScriptableObject**
  - Конфіги та параметри ігрових сутностей зберігаються в ассетах, що спрощує баланс та розширення контенту

## Структура проєкту

Ключова структура знаходиться в `Assets/`:

- **`Assets/GameResources/`**
  - **`Configs/`**
    - ScriptableObject-конфіги (параметри ворогів/зброї/куль/тощо)
  - **`Game/`**
    - Ігрові ресурси (префаби, графіка, тощо) які використовуються під час гри
  - **`Scenes/`**
    - Сцени Unity

- **`Assets/Scripts/`**
  - **`ConfigData/`**
    - Моделі/структури даних, що описують конфіги та їхні залежності
  - **`Core/`**
    - Базова інфраструктура: загальні сервіси/утиліти/контракти
  - **`GameServices/`**
    - Ігрові сервіси та системи (спавн, менеджери, інтеграція конфігів та фабрик)

- **`Assets/Plugins/`**
  - Плагіни, зокрема **Zenject** та **DOTween** (Demigiant)

## Додавання нового контенту (вороги / турелі / кулі)

Проєкт підтримує розширення через конфіги на базі `ScriptableObject`.

Загальна ідея:

- **Створюєш новий конфіг-ассет** у `Assets/GameResources/Configs/` для потрібного типу сутності.
- **Додаєш/налаштовуєш префаб** (спрайти, колайдери, анімації за потреби) та лінкуєш його з конфігом.
- **Реєструєш** (якщо потрібно) тип у відповідній фабриці/інсталері Zenject, щоб спавн/створення відбувались через DI.

Очікуваний результат: новий ворог/турель/куля підхоплюється системами гри через конфіг без зміни основної логіки.

## В планах

- **Розбити конфіг на складові** (більш модульна структура ScriptableObject-конфігів):
  - Окремий конфіг для **рівнів**
  - Окремий конфіг для **машин гравця**
  - Окремий конфіг для **ігрових персонажів** (вороги/союзники/нейтральні)
